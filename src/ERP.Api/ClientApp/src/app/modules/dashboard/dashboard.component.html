Validacion Cross-Site Request Forgery (XSRF/CSRF)

Los tokens antiforgery están diseñados para proteger contra ataques CSRF y generalmente se aplican a solicitudes que cambian el estado del servidor, como POST, PUT, DELETE, etc. Las solicitudes GET, por su naturaleza, deberían ser idempotentes y no cambiar el estado del servidor, por lo que normalmente no se protegen con tokens antiforgery.
